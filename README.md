# Cloud Fitness & Progressive Overload Tracker

Backend API for tracking workouts, exercises, and progressive overload.

## Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- Hibernate
- H2 (local)
- PostgreSQL (Azure)
- Flyway
- Azure App Service (planned)

## Features
- Workouts with date & time
- Exercise & muscle group modelling
- Set-level tracking (reps, weight, RIR)
- Progressive overload (last set per exercise)
- Analytics-ready data model

## Status
- Phase 1 backend complete
- Azure deployment in progress
